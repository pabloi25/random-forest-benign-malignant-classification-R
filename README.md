# Random Forest for Benign/Malignant Classification

## Overview
This project applies a Random Forest model in R to classify samples as benign or malignant using supervised machine learning.

## Objectives
- Load and explore the dataset.
- Train a Random Forest classifier.
- Evaluate model performance using accuracy, sensitivity, specificity and confusion matrix.
- Identify the most relevant variables for classification.

## Tools
- R
- randomForest
- caret
- ggplot2
- dplyr

## Workflow
1. Data loading
2. Data preprocessing
3. Train/test split
4. Model training
5. Model evaluation
6. Variable importance analysis

## Results
The model achieved good classification performance and allowed identification of the most informative variables for distinguishing benign and malignant samples.
